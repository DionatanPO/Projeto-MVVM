Tech Stack & Arquitetura
Este projeto foi desenvolvido utilizando Flutter (Dart 3+) com foco em escalabilidade, manutenibilidade e suporte multiplataforma (Mobile e Desktop). A engenharia do software segue rigorosamente os padrões de mercado para 2025:
Arquitetura: Clean Architecture (Feature-First) + MVVM.
Gerenciamento de Estado: Riverpod com Code Generation (@riverpod).
UI System: Material 3 Expressive, layouts responsivos/adaptativos e temas customizados.
Navegação: GoRouter (Roteamento declarativo).
